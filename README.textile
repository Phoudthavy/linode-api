h1. "Linode API (Java)":http://github.com/theo/linode-api

Linode API client written in Java


h2. Build

The project can be built using Maven2 ("http://maven.apache.org/":http://maven.apache.org/).
Execute:
mvn clean package

h2. Documentation

See "JavaDocs":http://notedpath.com/projects/linode-api/javadocs/
Alternatively, run mvn javadoc:javadoc
